## Gitbook
```bash
Gitbook是使用NPM来进行安装的，可以在命令行中输入下面的命令进行安装：
$ npm install gitbook -g
让gitbook支持命令行：
$ npm install -g gitbook-cli
检验是否安装成功:
$ gitbook -V
1.5.0
初始化一本书
$ gitbook init
安装插件 插件配置在book.json中
$ gitbook install
开启服务
$ gitbook serve
```
